# Signalrausch Webapp

Diese Webapp dient als Trainingsplaner für das Signalrausch-Projekt.

## Installation

```bash
npm install
npm run dev
```

## Produktion

```bash
npm run build
npm start
```

## Features

- Zufällige Tagesplan-Generierung
- Kleiderschrank-Verwaltung
- Fortschrittsübersicht
- Erfolge mit Symbolen
